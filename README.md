# 待办事项 CLI 小工具

一个简单的命令行待办事项管理工具，支持按日期自动归档。

## 快速开始

```bash
pip install win10toast
python todo.py
```

## 使用

| 命令 | 说明 |
|------|------|
| `add <内容>` | 添加待办事项，自动归入今天 |
| `list` | 列出今天的所有待办 |
| `list 2026-05-16` | 查看指定日期的待办 |
| `done <id>` | 标记完成 |
| `delete <id>` | 删除事项 |
| `yesterday` | 查看昨天未完成的事项 |
| `quit` | 退出 |

## 数据存储

所有待办保存在 `todo.json`，按日期分组。每天的事项自动归档，历史记录不会被覆盖。

## 开机自启

1. 按 `Win + R`，输入 `shell:startup`，回车
2. 将 `todo.py` 或它的快捷方式放入打开的文件夹
